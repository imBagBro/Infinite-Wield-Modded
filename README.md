# Infinite Yield FE - Versão Modificada

Versão modificada do Infinite Yield FE original criado por EdgeIY.

## Modificações
- Comando `savegame` agora salva os avatares, tools e itens 
  dos jogadores presentes no servidor junto com o mapa

## Como usar
```lua
loadstring(game:HttpGet('SEU_LINK_RAW_AQUI'))()
```
